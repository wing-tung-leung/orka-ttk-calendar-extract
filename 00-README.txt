Vereisten:
----------
1 - Standard unix text tools
2 - Python interpreter
3 - internet connectie

In't algemeen zijn deze makkelijk terug te vinden op een standaard Linux
desktop installatie. Met cygwin op MS Windows werkt het waarschijnlijk
ook wel.


Te volgen stappen:
------------------

1 - download all pages (senior + junior): use download script named
    "download-orka-calendars.sh"

2 - Controleer of sommige Orka ploegen samenvallen in 1 reeks. Indien ja,
    1 van de twee HTML bestanden verwijderen om dubbels te voorkomen.

3 - Verwerk all kalenderbestanden van seniors: process.sh

     Opletten voor ploegnamen, kan veranderen: "Orka", "ORKA", "Orka TTK", ..

4 - Controleer aantal lijnen van "all_games.csv": normaal ongeveer 20/22
    matchen per ploeg in reeks, dus voor 12 ploegen ongeveer 240/264 lijnen.


Gelijkaardig process voor jeugd. Formaat jeugd licht anders.
